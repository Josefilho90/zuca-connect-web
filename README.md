# Zuca Couros Front-end

Sistema de gestão comercial construído com Angular standalone, rotas lazy, signals, Reactive Forms e persistência em `localStorage`.

## Executar

```bash
npm install
npm start
```

Acesse `http://localhost:4200`.

## Versão do Angular

O projeto usa a versão oficial mais recente disponível no registro durante a criação (`21.x`). O Angular 22 ainda não estava publicado. A arquitetura usa APIs modernas e não depende de APIs obsoletas, mantendo a futura atualização concentrada nas dependências do `package.json` e no comando `ng update`.
